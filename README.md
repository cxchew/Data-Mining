# Data Mining (SECP2753) 🍄📊🧠

Welcome to my portfolio repository for **SECP2753: Data Mining**. This repository documents my hands-on experience and theoretical understanding of the complete knowledge discovery process—spanning exploratory data profiling, data pre-processing, pattern extraction, and model performance validation.

The core focus of this course is learning how to transform noisy, uncurated real-world datasets into valuable strategic insights by designing, optimizing, and deploying both supervised and unsupervised machine learning models.

## 🛠️ Core Technological & Theoretical Toolkit

* **Data Profiling & Pre-processing Frameworks:** Missing Value Imputation, Inconsistent String Normalization, Outlier Filtering, Correlation Heatmap Tracking.
* **Feature Engineering & Transformation:** Min-Max Feature Scaling, Equal-Width Binning, Equal-Frequency Discretisation, Categorical Multi-Class Label Encoding.
* **Pattern Recognition & Rule Mining:** Apriori Frequency Rules Filtering, Market Basket Association Mapping, Metric Optimization (Support, Confidence, Lift).
* **Supervised Learning Models:** Logistic Regression, Decision Tree Classifiers, Random Forest Ensembles, Split Partition Cross-Validation.
* **Unsupervised Clustering Frameworks:** Centroid Allocation K-Means, Agglomerative Hierarchical Trees, Dendrogram Linkage Synthesis, Elbow Method WCSS Analysis.
